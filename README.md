# SecureGram — Client

P2P зашифрованный мессенджер. Работает прямо в браузере, не требует установки.

## Деплой на GitHub Pages (5 минут)

### 1. Создай репозиторий

- Зайди на github.com → New repository
- Имя: `securegram` (или любое)
- Public ✓
- Create repository

### 2. Залей файлы

Через браузер (самый простой способ):
- Открой репозиторий → Add file → Upload files
- Перетащи `index.html` и `manifest.json`
- Commit changes

### 3. Включи GitHub Pages

- Settings → Pages
- Source: **Deploy from a branch**
- Branch: **main** → / (root)
- Save

Через минуту появится ссылка вида:
```
https://ТВО-ИМЯ.github.io/securegram/
```

### 4. Подключи свой сигнальный сервер (после деплоя на Railway)

Открой `index.html`, найди строку:
```js
// { host: 'YOUR-RAILWAY-DOMAIN.up.railway.app', ...
```
Раскомментируй и замени домен на свой. Загрузи файл снова.

## Использование

1. Открой ссылку в браузере
2. Нажми на свой ID чтобы скопировать
3. Отправь ID другу любым способом
4. Друг вставляет твой ID → Добавить
5. Соединение P2P, ключи обменялись — можно писать

## Что внутри

- **WebRTC** — прямое P2P соединение между браузерами
- **ECDH P-384** — обмен ключами при подключении
- **AES-256-GCM** — шифрование каждого сообщения
- **Mesh** — сообщения автоматически доходят через других участников
- **PWA** — устанавливается на телефон как приложение
