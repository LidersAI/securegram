# LIDERS CHAT — Branding Guide

## Concept: "Shadow Logic"
A blend of classic intellect (Gothic L) and future technology.
Represents security, secrecy, and high complexity.

## Color Palette

| Name | Hex | Usage |
|---|---|---|
| Sapphire Blue | `#103060` | Primary background, buttons |
| Deep Navy | `#0d1a2e` | App background |
| Silver Metallic | `#B0C0D0` | Gothic L symbol |
| Cyan Glow | `#30D5F5` | Accent, highlights, links |
| Ice White | `#F0F4F8` | Text on dark |

## Typography
- **Display:** SF Pro Display / Helvetica Neue (system)
- **Monospace:** SF Mono / Fira Code (IDs, codes)
- **Note:** Gothic font is used ONLY in the L symbol — never for body text

## Assets

| File | Usage |
|---|---|
| `icon-512.png` | App store, PWA |
| `icon-192.png` | Android home screen |
| `icon-192-maskable.png` | Android adaptive icon |
| `favicon.ico` | Browser tab |

## Icon Usage Rules
- Always use the Gothic L on Sapphire Blue background
- Minimum size: 32×32px
- Do not change colors or add effects
- Do not use Gothic font for any text labels

## Voice & Tone
- **Secure** — emphasize encryption and privacy
- **Premium** — professional, not playful
- **Simple** — complex tech, simple UX
